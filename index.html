    • 1.1 Conceptos básicos de la estadistica

  • Definición Teoría de decisión:

    • es un enfoque que busca proporcionar un marco para la toma de decisiones en situaciones de incertidumbre. Esta teoría se basa en la idea de que las decisiones se toman en función de la información disponible y las posibles consecuencias de cada opción

  • Definición población:

    • la población se refiere al conjunto total de elementos o individuos que comparten características similares y que son objeto de estudio en un análisis estadístico. Esta población puede ser cualquier grupo de interés, como personas, objetos, eventos, u otras entidades que comparten ciertas propiedades que se desean estudiar.

  • Definición Teoría de decisión:

    • se refiere a un proceso que permite seleccionar una muestra de una población de forma aleatoria, donde cada individuo tiene la misma probabilidad de ser elegido. Este método se utiliza para obtener datos representativos de la población y así poder realizar análisis estadísticos más precisos.

  • Definición Parámetros aleatorios.:

    • se refieren a las características o propiedades que varían de forma aleatoria en una población. Estos parámetros pueden ser variables como la edad, altura, peso, ingresos, entre otros, que pueden ser medidos o cuantificados en cada individuo de la población.

2.1 Técnicas de Conteo

  • Principio Auditivo:

    • El Principio aditivo es una técnica de conteo que consiste en sumar el número de formas en que se puede realizar un evento o una tarea, considerando que estas formas son mutuamente excluyentes.

  • Principio multiplicativo:

    • El Principio multiplicativo es una técnica de conteo que se basa en multiplicar el número de opciones disponibles para cada etapa de un proceso para determinar el número total de resultados posibles.

  • Notación Factorial:

    • La Notación Factorial es una técnica de conteo que se expresa con el símbolo "!". Se utiliza para denotar el producto de todos los enteros positivos menores o iguales a un número dado.

  • Principio multiplicativo:

    • la población se refiere al conjunto total de elementos o individuos que comparten características similares y que son objeto de estudio en un análisis estadístico. Esta población puede ser cualquier grupo de interés, como personas, objetos, eventos, u otras entidades que comparten ciertas propiedades que se desean estudiar.

  • Permutaciones:

    • Las Permutaciones son arreglos ordenados de elementos donde el orden importa. Se utilizan para contar el número de formas en que se pueden ordenar los elementos de un conjunto.

  • Combinaciones:

    • Las Combinaciones son selecciones no ordenadas de elementos donde el orden no importa. Se utilizan para contar el número de formas en que se pueden seleccionar elementos de un conjunto sin importar el orden.

  • Diagrama de Árbol:

    • El Diagrama de Árbol es una representación gráfica utilizada para visualizar y contar todas las posibles ramificaciones y resultados de una serie de decisiones o eventos secuenciales.

  • Teorema del Binomio:

    • El Teorema del Binomio es una fórmula que permite expandir un binomio elevado a una potencia determinada. Es útil para calcular coeficientes de términos específicos en la expansión de un binomio.

    2.2 Teoría elemental de probabilidad.

    La teoría elemental de probabilidad es la base para entender eventos aleatorios. Incluye conceptos fundamentales como espacio muestral, eventos y cálculo de probabilidades. Ayuda a medir la posibilidad de que ocurra un evento y es esencial en estadística y toma de decisiones.
